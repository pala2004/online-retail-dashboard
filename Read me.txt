Online Retail Data Visualization
This project contains a Tableau dashboard and dataset focused on analyzing online retail sales. The visualization provides insights into customer behavior, product performance, sales trends, and regional distribution to support business decision-making.

Project Structure:
Online-Retail-Visualization/
|-- Online Retail (1).xlsx                  -> Dataset used for analysis
|-- online retail data visuvalization1.twbx -> Tableau packaged workbook
|-- README.txt                              -> Project documentation

Dashboard Overview:
The Tableau dashboard includes:
- Top-selling products by quantity and revenue
- Monthly sales trends
- Country-wise sales breakdown
- Customer segmentation
- Key metrics like total revenue and repeat customer rate

To view the dashboard:
- Open the .twbx file in Tableau Desktop
- OR visit the Tableau Public link (if available)

Tableau Public Link:
(View here if published): https://public.tableau.com/your-dashboard-link

Dataset Information:
Source: UCI Machine Learning Repository
File: Online Retail (1).xlsx
Main Columns:
- InvoiceNo
- StockCode
- Description
- Quantity
- InvoiceDate
- UnitPrice
- CustomerID
- Country

Tools Used:

How to Use:
